# Software Projects

This document contains expanded case studies for selected software products developed by John Dadzie through Airmonlink.

## Project Status Definitions

- **Planned:** Product requirements and architecture are being prepared.
- **Source developed:** The implementation exists in source code.
- **Source validated:** Formatting, static analysis or tests have been completed.
- **Release built:** A distributable build has been compiled successfully.
- **Device tested:** The application has been tested on a physical device.
- **Production deployed:** The system has been deployed for active use.
